# ancient-changan-digital-twin
# 两京·策—基于多源数据融合的历史地理信息空间决策可视化平台
## 项目定位
基于多源数据融合的空间智能交互系统，适配数据科学与数字孪生应用，实现「数据采集-分析-决策-迭代」全闭环。

## 技术栈
- 前端：UE5 + Cesium（地理空间）+ MetaHuman数字人
- 服务端：Linux + C++ 轻量化HTTP服务
- AI能力：DeepSeek大模型 + 腾讯云语音ASR/TTS
- 数据层：SQLite + Python数据分析（Pandas/Matplotlib）
- 部署：本地虚拟机、数据驱动决策系统

## 核心功能
1. 多模态数据采集（空间地理/语音交互/用户行为）
2. 数据驱动场景自适应优化（解决Cesium卡顿）
3. 数字人智能对话 + 地理语义联动
4. 离线数据分析与系统自迭代

## 学术价值
贴合数据科学：多源异构数据融合、空间数据科学、数据驱动决策
